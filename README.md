# SW_Lab_Assignment_4

## Software Engineering Lab – Assignment 4

**Topic:** Version Control Using Git

This repository contains three mini projects implemented in **Python** with **CSV files used as data storage**.
The projects demonstrate simple management systems while using **Git and GitHub for version control and collaboration**.

---

# Projects Included

## 1. Inventory Management System

This program manages store products and inventory.

### Features

* Add new products
* Update product details
* View product list
* Store product data in CSV

### Files

* `A2_07_4_1_55.py` – Python program
* `A2_07_4_1_55.csv` – Product data

---

## 2. Student Marks Management System

A system to manage and update student marks for different subjects.

### Features

* Store student names and roll numbers
* Update subject marks
* Store marks in CSV format
* Display marks in tabular format

### Files

* `A2_07_4_2_41.py` – Python program
* `A2_07_4_2_41.csv` – Student marks database

---

## 3. Task Management CLI Tool

A command-line tool for managing tasks.

### Features

* Add tasks
* Update tasks
* Mark tasks as completed
* Maintain task records using CSV

### Files

* `A2_07_4_3_59.py` – Python program
* `A2_07_4_3_59.csv` – Task database

---

# Technologies Used

* Python
* CSV for data storage
* Git for version control
* GitHub for repository hosting

---

# How to Run

Clone the repository:

```
git clone https://github.com/<your-username>/SW_Lab_Assignment_4.git
```

Navigate to the project folder:

```
cd SW_Lab_Assignment_4
```

Run any of the programs:

```
python A2_07_4_1_55.py
python A2_07_4_2_41.py
python A2_07_4_3_59.py
```
